# Plano 07-01: Consolidação de Financeiro, Pendências e Documentos

Este plano foca em completar a Fase 7 do Roadmap, garantindo que o Diego possa gerenciar cobranças, pendências e, principalmente, entregar documentos finais de forma organizada.

## Objetivos
1. **Entrega de Documentos (ADM-10):** Criar uma seção dedicada para upload de documentos "entregáveis" (plantas, alvarás, etc) que o cliente possa baixar facilmente.
2. **Organização do Painel de Arquivos:** Melhorar a aba de arquivos no Admin para gerenciar tanto o Google Drive quanto os arquivos locais.
3. **Melhoria da Experiência do Cliente:** Exibir os documentos entregues de forma proeminente na área do cliente.

## Tarefas

### 1. Banco de Dados e Schema
- [x] Atualizar `area-cliente/includes/schema.php` para incluir a tabela `processo_entregaveis`.
  ```sql
  CREATE TABLE IF NOT EXISTS processo_entregaveis (
      id INT AUTO_INCREMENT PRIMARY KEY,
      cliente_id INT NOT NULL,
      titulo VARCHAR(255) NOT NULL,
      arquivo_path VARCHAR(255) NOT NULL,
      data_upload DATETIME DEFAULT CURRENT_TIMESTAMP,
      FOREIGN KEY (cliente_id) REFERENCES clientes(id) ON DELETE CASCADE
  );
  ```

### 2. Admin: Gestão de Entregáveis
- [x] Modificar `area-cliente/includes/views/admin/arquivos.php` para incluir a lista de entregáveis e um formulário de upload.
- [x] Criar `area-cliente/actions/admin/entregavel_upload.php` para processar o upload.
- [x] Criar `area-cliente/actions/admin/entregavel_delete.php` para excluir arquivos.

### 3. Portal do Cliente: Visualização de Documentos
- [x] Modificar `area-cliente/client-app/documentos.php` para listar os arquivos da tabela `processo_entregaveis`.
- [x] Estilizar a lista de documentos para parecer uma "central de downloads".

### 4. Polimento Financeiro e Pendências
- [x] Revisar `area-cliente/includes/views/admin/financeiro.php` e `pendencias.php` para garantir que as mensagens de sucesso/erro estão funcionando.
- [x] Validar a integração com o WhatsApp para cobranças.

## Critérios de Sucesso
1. Diego consegue subir um PDF de "Alvará" na aba Arquivos do cliente.
2. O arquivo aparece na lista com opção de excluir.
3. O cliente logado vê o "Alvará" na sua aba de Documentos e consegue baixar.
4. Lançamentos financeiros e pendências continuam funcionando perfeitamente.
